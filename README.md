#  Jogo da Velha

Um jogo da velha moderno com visual neon, desenvolvido com **HTML, CSS e JavaScript puro**.

---

## Preview
Interface com estilo neon, animações e modo contra robô.

---

##  Funcionalidades
- Modo 2 jogadores
- Modo contra robô (IA simples)
- Placar automático
- Animações suaves
- Botão para reiniciar a partida
- Estilo moderno "Neon UI"

---

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ▶ Como executar o projeto

1. Clone o repositório:
```bash
git clone https://github.com/analacerda-dev/jogo-da-velha.git
```

2. Entre na pasta do projeto:
```bash
cd jogo-da-velha
```

3. Abra o arquivo no navegador:
```bash
open index.html
```
ou simplesmente dê dois cliques no `index.html`.

---

##  Objetivo do projeto

- Praticar lógica de programação
- Aprender manipulação do DOM
- Criar jogos simples com JavaScript
- Trabalhar com animações e interface UI
